# 牛轉門：時空導航儀（終極完全版）

多週期共振路徑預測與門控轉態偵測的可視化工具。以月/週/日到 1m 八層時框，結合「7 字訣」「門控（W/M）」「波動標籤（1/2/3/A/B/C）」模擬結構演化，提供趨勢評分、情境預言、風險控管與極端行情警示。此版為「終極完全版」前端單頁，包含強化的解說區、警示模組、範例載入與視覺優化。

## 核心概念

- 波動標籤：1、2、3、A、B、C
  - 1/3/B：正弦相位（偏漲，紅）
  - 2/A/C：餘弦相位（偏跌，綠）
- 7 字訣方向：UP（紅等黑）／DOWN（黑等紅）
  - 對 B 段方向進行語意修正（在評分中影響多空向量）
- 門控（Gate）
  - 門屬性：W 門（底）／M 門（頭）／無門
  - 門開關：形成中／已開啟／已關閉
  - 收盤確認：所有門控判讀以該時框收盤價為準

## 運作邏輯（簡述）

- 時框參數
  - timeframes 包含每層級的 weight（能量權重）與 ratio（步進節奏）
- 多空評分
  - 1/3/B 視為 +1，2/A/C 視為 -1；B 並受 7 字訣修正（DOWN 時視為 -1）
  - 以各層級方向 × 權重相加，> 20 多頭慣性；< -20 空頭重力；否則中性整理
- 狀態推演
  - 依起始標籤與 ratio 做步進輪替，生成每一步各時框標籤
- 極端警示
  - 宏觀（前 5 層：月→60m）若全為 3 或全為 C 即觸發提示
  - 顯示微觀（15m/5m/1m）門開/關計數
- 能量等級
  - 以 totalPower（上行標籤加權）分級：1 極多、2 強多、3 多空關鍵、4 空誘多、5 空頭
- 門控文案
  - 依參考層級當前標籤映射為：開門／測門／B 反彈空點／關門

## 安裝與部署

- 檔案組成：單一 HTML（含 CSS/JS），例如 `index.html`
- 快速使用：
  1) 將提供之 HTML 貼到 `index.html`
  2) 瀏覽器直接開啟或以任一靜態伺服器部署（Nginx/Apache/Vercel/Netlify/Cloudflare Pages）
- HTTPS 建議：對外服務請設反向代理（Nginx）+ Let’s Encrypt 憑證，強制 HTTPS
- PWA 提示：已內嵌 manifest；如需離線能力，另行加入 Service Worker 與快取策略

## 使用步驟

1. 開啟 `index.html`
2. 點擊「載入範例」或於設定面板針對每層級設定：
   - 波動標籤：1/2/3/A/B/C
   - 7 字訣方向：UP / DOWN
   - 門屬性：無門 / W 門 / M 門
   - 門開關：形成中 / 已開啟 / 已關閉
3. 在控制區選擇：
   - 導航基準層級（預設 4H）
   - 預演步數（預設 60）
4. 點擊「執行多週期矩陣推演」
5. 觀察：
   - 儀表板：多空評分、Scenario、Defense Point
   - 警示區：是否出現 33333 / CCCCC 結構與微觀門統計
   - 結果表格：各時框標籤演化、能量等級與門控提示
6. 注意：一切以「該時框收盤價」確認有效

## 自訂與擴充

- 權重/節奏：調整 `timeframes` 的 `weight` 與 `ratio` 以符合你的策略
- 情境規則：擴充 Scenario 判斷與極端警示條件（如多組宏觀/微觀組合）
- 視覺樣式：修改 `<style>` 變數（主題色、標籤色階、卡片樣式）
- 門控邏輯：將簡化的標籤輪替替換為更完整的破門/偽突破/回抽測門演算法
- 資料接入：串行情 API 或你的內部資料庫；加入狀態保存（LocalStorage/IndexedDB）

## 常見問題

- 為何出現 33333 / CCCCC 警示？
  - 代表宏觀結構極端一致，風險/轉折概率提升；請配合微觀門開關數量判讀
- 趨勢評分如何計算？
  - 各層級標籤方向 × 權重相加（B 段受 7 字訣修正），> 20 多頭慣性、< -20 空頭重力
- 門控提示來源？
  - 依參考層級當前標籤映射為開門／測門／B 反彈空點／關門
